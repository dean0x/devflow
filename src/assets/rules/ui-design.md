# UI Design

**Every aesthetic choice must have intent — no decoration without purpose.**

- Design tokens over magic numbers — spacing, color, typography from system
- 4.5:1 minimum contrast ratio for text (WCAG AA)
- 8px grid for spacing consistency
- Mobile-first responsive: start small, enhance upward
