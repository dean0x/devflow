## Architecture & Design

### Key Architectural Decisions

**Command-Level Orchestration Pattern**:
Commands run in main context with Task tool access. Agents cannot spawn other agents. This means:
- `/specify` spawns Explore + Plan agents directly
- `/swarm` spawns Explore + Plan + Coder + Review agents directly
- No multi-wave orchestration (why `/plan` and `/coordinate` were removed)

**Clear Separation of Concerns**:
| Command | Focus | Agents | Output |
|---------|-------|--------|--------|
| `/specify` | Requirements (what & why) | 4 Explore + 3 Plan | GitHub issue |
| `/swarm` | Implementation (how) | 4 Explore + 3 Plan + N Coder + Reviews | PR |
