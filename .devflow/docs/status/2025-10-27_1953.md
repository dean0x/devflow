## 🏗️ Architecture & Design

### Key Architectural Decisions

1. **Shared Utilities Extraction** (v0.5.0)
   - Created `src/cli/utils/paths.ts` and `src/cli/utils/git.ts`
   - Eliminated ~65 lines of code duplication between init and uninstall
   - Single source of truth for path resolution and git operations

2. **Async Git Operations**
   - Replaced blocking `execSync` with `promisify(exec)`
   - Non-blocking I/O improves CLI responsiveness
   - Maintains compatibility across cgroup v1/v2 and bare metal

3. **Atomic File Operations**
   - Fixed TOCTOU race conditions using `fs.writeFile` with `'wx'` flag (exclusive create)
   - Applied to: settings.json, CLAUDE.md, .claudeignore
   - Prevents accidental overwrites during concurrent operations

4. **Installation Scope Pattern**
   - Two-tier installation: user-wide vs project-specific
   - Interactive prompts with TTY detection for CI/CD compatibility
   - Graceful fallback to defaults in non-interactive environments
