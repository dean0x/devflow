## Architecture & Design

### Key Architectural Decisions
1. **Tiered Skills System**:
   - Tier 1: Foundation skills (shared by agents)
   - Tier 2: Specialized skills (auto-activate)
   - Tier 3: Domain-specific (language/framework)

2. **Unified Reviewer Pattern**:
   - Single agent with `FOCUS_AREA` and `PATTERN_SKILL` prompt injection
   - Loads appropriate pattern skill based on focus
   - Consistent 6-step review methodology across all focuses

3. **Self-Review Framework**:
   - 9 pillars: Design, Functionality, Security (P0), Complexity, Error Handling, Tests (P1), Naming, Consistency, Documentation (P2)
   - Enforced via Stop hook in Coder agent
   - Must pass all P0 and P1 before returning
