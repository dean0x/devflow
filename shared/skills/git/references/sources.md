# Git & GitHub — Sources

Bibliography for the `git` skill.

---

## References

1. **Pro Git** — Chacon & Straub (2014). Authoritative git internals, branching model, index/lock mechanics. [git-scm.com/book](https://git-scm.com/book/en/v2)
2. **Conventional Commits 1.0.0** — Specification for structured commit messages with type prefixes. [conventionalcommits.org](https://www.conventionalcommits.org/en/v1.0.0/)
3. **Git Documentation** — Official reference for `git-lock`, `git-reset`, `git-rebase`, `git-stash`. [git-scm.com/docs](https://git-scm.com/docs)
4. **GitHub REST API** — Endpoints for pulls, issues, releases, rate limits. [docs.github.com/rest](https://docs.github.com/en/rest)
5. **GitHub GraphQL API** — Schema for efficient queries and mutations. [docs.github.com/graphql](https://docs.github.com/en/graphql)
6. **Semantic Versioning 2.0.0** — Version format specification (MAJOR.MINOR.PATCH). [semver.org](https://semver.org/)
7. **gh CLI Manual** — GitHub's official CLI for PR, issue, release, and workflow operations. [cli.github.com/manual](https://cli.github.com/manual/)
8. **OWASP Secrets Management** — Patterns for preventing credential leaks in version control. [cheatsheetseries.owasp.org](https://cheatsheetseries.owasp.org/cheatsheets/Secrets_Management_Cheat_Sheet.html)
