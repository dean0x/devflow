# Accessibility

**Every interaction must work without a mouse.**

- Semantic HTML first — ARIA only when native elements won't do
- Focus management on route changes and dynamic content
- Color is never the only indicator — use text, icons, or patterns
- All images need alt text; decorative images get `alt=""`
